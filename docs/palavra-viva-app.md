# Palavra Viva — Plano de Desenvolvimento

> **"Onde a Fé Encontra a Tecnologia"**
> Appi Company © 2026

> Plano completo disponível no artifact da conversa.
> Este arquivo serve como referência rápida no projeto.

## Stack: Flutter + Riverpod + GoRouter + Hive + Dio + OpenAI

## MVP Features
1. **Palavra Viva** — Chat com IA teológica
2. **Bíblia Interativa** — Exploração de versículos
3. **Orações Personalizadas** — Geração baseada no sentimento
4. **Diário Espiritual** — Registro da jornada de fé

## Design System: "Celestial Dark"
- Background: #0A0E1A | Primary: #D4A853 (gold) | Secondary: #5B8DEF (celestial blue)
- Fonts: Outfit (UI) + Merriweather (bíblicos)
- Dark mode padrão, OLED-optimized

## Navigation: Bottom Tab Bar (4 tabs)
🏠 Home | 📖 Bíblia | 🙏 Orar | 📓 Diário

## Fases
- **Fase 1 (P0):** Foundation — projeto, deps, theme, widgets, router
- **Fase 2 (P1):** Core — home, chat IA, bíblia, orações, diário
- **Fase 3 (P2):** Polish — splash, animações, configurações
- **Phase X:** Verificação final (builds, performance, a11y)
